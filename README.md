# Project Checkmate

Project Checkmate (final name TBD) is an innovative take on the classic game of Checkers, where players will have the added challenge of navigating their pieces through a minefield.

## Usage

The game, when released, will be available on itch.io.  A release is expected by mid-August and will run on Windows, MacOS and Linux.
